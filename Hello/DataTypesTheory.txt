Variable - это именованная область памяти,
содержимое которой может изменяться 
в процессе выполнения программы.

		type name;
type - тип переменной. Определяет:
	1. Сколько памяти 
	2.
	3.

  I) Лоические типы: bool (Boolean)
	true	1
	false	0
	0 - это false, а true - это все что НЕ 0.
 II) Символьные типы: char (Character - Символ).
	  ASCII - American Standard Code for Information Interchange
	  256

III) Числовые типы

	  signed - могут хранить как положительные так и отрицательные целые числа
	unsigned - предназначены для хранения лишь положительных целых чисел

	int a;	//она сможет хранить как положительные так и отрицательные целые числа
	unsigned int b;	//объявление беззнаковой переменной

	short		2B
	long		4B
	int			4B
	long long	8B
